# Решение Практикума №1 — Методы сортировки (Bubble Sort и Shaker Sort)

Реализация двух методов сортировки и экспериментальное сравнение их эффективности на массивах разной длины и типа.

## Структура проекта

```
buklov/
├── src/
│   ├── main.c           - главная программа с экспериментами
│   ├── sorting.h        - заголовок с объявлениями функций сортировки
│   ├── bubble-sort.c    - реализация пузырьковой сортировки
│   ├── shaker-sort.c    - реализация сортировки коктейльными пузырьками
│   ├── generator.h      - заголовок генератора массивов
│   └── generator.c      - генератор тестовых массивов
├── report/
│   ├── report.tex       - отчёт в формате LaTeX
│   └── report.pdf       - скомпилированный отчёт
└── README.md            - этот файл
```

## Сборка проекта

### Требования
- GCC (или другой C компилятор)
- Флаги: `-O2 -Wall -Wextra -std=c11`

### Компиляция

```bash
gcc -O2 -Wall -Wextra -std=c11 src/*.c -o lab1
```

Или если вы находитесь в папке `buklov/`:

```bash
gcc -O2 -Wall -Wextra -std=c11 src/main.c src/bubble-sort.c src/shaker-sort.c src/generator.c -o lab1
```

## Запуск программы

```bash
./lab1
```

Программа проводит экспериментов с размерами массива:
- **n = 10, 100, 1000, 10000**

Для каждого размера создаются **4 типа массивов**:
1. Упорядочен по возрастанию
2. Упорядочен по убыванию
3. Случайный массив
4. Случайный массив (другой диапазон)

Каждый эксперимент повторяется **5 раз** и выводятся результаты:
- Количество **сравнений** элементов
- Количество **перемещений** элементов

Результаты выводятся в виде таблиц для каждого типа массива.

## Реализованные алгоритмы

### Пузырьковая сортировка (Bubble Sort)
Простой алгоритм, многократно проходящий по массиву и обменивающий соседние элементы, если они расположены в неправильном порядке. Имеет временную сложность O(n²) в среднем случае.

### Сортировка коктейльными пузырьками (Shaker Sort)
Модификация пузырьковой сортировки, проходящая по массиву поочередно слева направо и справа налево. Может быть более эффективной на некоторых типах данных (частично упорядоченные массивы).

## Функции программы

- `generate_array(n, arr, type)` — генерирует массив размера `n` указанного типа
- `bubble_sort(n, arr)` — сортирует массив пузырьковым методом
- `shaker_sort(n, arr)` — сортирует массив методом коктейльных пузырьков
- `swap(a, b)` — вспомогательная функция обмена элементов

## Результаты и интерпретация

Программа позволяет:
- Сравнить эффективность двух алгоритмов на разных типах данных
- Наблюдать, как количество операций растёт с размером массива
- Оценить влияние начального порядка элементов на производительность алгоритмов

Полный анализ и графики результатов находятся в файле **report/report.pdf**.

## Примечания

- Все эксперименты используют одни и те же исходные массивы для корректного сравнения алгоритмов
- Измеряются как сравнения, так и перемещения для точной оценки эффективности
- Программа использует `srand(time(NULL))` для инициализации генератора случайных чисел
